1) Install Anaconda
2) Create the python environment
		conda update conda
		conda create --name planet-b python=3.5 #planet-b is just a name
3) Activate the Anaconda environment
		activate planet-b
4) Clone the planet-b repo
5) Inside of the root folder of the repo and if you're on windows
		requirements.in 
			comment out   "uwsgi>=2.0.0"
		requirements.txt  "uwsgi==2.0.15"
